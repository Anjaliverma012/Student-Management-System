# 🎓 Student Management System

A responsive Student Management System developed using **Django**, **Python**, **SQLite**, **HTML**, **CSS**, and **Bootstrap**. The project allows users to manage student records with complete CRUD operations.

---

## 🚀 Features

- ➕ Add Student
- 📋 View Student List
- ✏️ Edit Student Details
- 🗑️ Delete Student
- 💻 Responsive Bootstrap UI
- 🗄️ SQLite Database
- 🔐 Django Admin Panel

---

## 🛠️ Technologies Used

- Python
- Django
- SQLite
- HTML5
- CSS3
- Bootstrap 5

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/anjaliverma012/Student-Management-System.git
```

2. Go to the project folder

```bash
cd Student-Management-System
```

3. Install dependencies

```bash
pip install django
```

4. Run migrations

```bash
python manage.py migrate
```

5. Start the server

```bash
python manage.py runserver


6. Open your browser
http://127.0.0.1:8000/
